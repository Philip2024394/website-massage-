You must add the bookingId attribute to your Appwrite bookings collection before creating any bookings. Without this attribute, all booking creations will fail with "Missing required attribute 'bookingId'" error.

‚úÖ Required Attributes Must Add

Attribute Name Type Size/Range Description
bookingId String 255 Unique booking identifier (auto-generated UUID or timestamp)
therapistId String 255 ID of the therapist or massage place
therapistName String 255 Display name of the therapist or place
therapistType String 50 Either 'therapist' or 'place'
duration Integer Min: 60, Max: 120 Service duration in minutes (60, 90, or 120)
price Integer Min: 0, Max: 1000 Service price in dollars
status String 50 Booking status: 'pending', 'confirmed', 'expired', 'completed'
createdAt DateTime - When the booking was created
responseDeadline DateTime - Therapist must respond before this time (5 or 15 minutes)

‚≠ê Optional Attributes Recommended

Attribute Name Type Size/Range Description
scheduledTime DateTime - For scheduled bookings - the future time slot selected by customer
customerName String 255 Customer's full name (collected in scheduled bookings)
customerWhatsApp String 50 Customer's WhatsApp number for communication
bookingType String 50 Either 'immediate' (5-min) or 'scheduled' (15-min)
providerId String 255 Alternative provider ID (if different from therapistId)
providerType String 50 Alternative provider type
hotelVillaId String 255 Hotel or villa ID if booking is made from venue live menu
roomNumber NEW String 50 Hotel/Villa room number for service delivery (e.g., "101", "205A")

üöÄ Quick Setup Instructions

  1. Go to Appwrite Console: https://cloud.appwrite.io/console
  2. Navigate to: Your Project ‚Üí Database ‚Üí bookings collection ‚Üí Attributes tab
  3. Click: "+ Create Attribute" button
  4. Add each attribute above with the exact name, type, and size
  5. Most Important: Add bookingId (String, 255) first!
  6. Wait: 1-2 minutes for Appwrite to sync the schema
  7. Test: Refresh your app and try creating a booking
