# Notification Sound Files Guide

## Critical: MP3 sound files required for bulletproof notifications

### Required Sound Files in `/public/sounds/`:

1. **booking-alert.mp3** - Loud, attention-grabbing for new bookings
2. **message-notification.mp3** - Moderate tone for messages  
3. **alert-urgent.mp3** - Very loud for critical alerts
4. **success-notification.mp3** - Pleasant confirmation sound

### Sound Characteristics:
- **Duration**: 2-3 seconds maximum
- **Volume**: Clear and audible
- **Format**: MP3 (best compatibility)
- **Frequency**: Mid-range tones (easier to hear)

### Recommended Sounds:
- **Booking Alert**: Bell chime or notification ding
- **Message**: Gentle notification tone
- **Urgent Alert**: Alarm-like sound
- **Success**: Pleasant confirmation chime

### Free Sound Resources:
- https://freesound.org/
- https://zapsplat.com/
- https://mixkit.co/free-sound-effects/
- Built-in system sounds

### Current Fallback:
The system will work without custom sounds but will use browser defaults.

### Testing:
Use the notification test function in the dashboard to verify all sounds work properly.
