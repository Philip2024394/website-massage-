# Auth App - Authentication & Membership Flows

This is the dedicated authentication micro-frontend application handling all user registration, login, and membership flows.

## Port
- Runs on: http://localhost:3001

## Features
- Member package selection (Pro/Plus)
- Account creation flow
- Terms & Conditions
- User authentication (Therapist, Massage Place, Facial Clinic)
- Privacy Policy

## Connected Dashboards
- Therapist Dashboard: http://localhost:3002
- Massage Place Dashboard: http://localhost:3005
- Facial Clinic Dashboard: http://localhost:3006
- Admin Dashboard: http://localhost:3004

## Development
```bash
npm install
npm run dev
```
