## 2. Home page / dropdown cities (Indonesia only)

The **home page city dropdown** and **filter** use a different source: `data/indonesianCities.ts` → `INDONESIAN_CITIES_CATEGORIZED` (and `ALL_INDONESIAN_CITIES`). That list has **locationId** (e.g. `yogyakarta`, `canggu`) and is used for:

- Filtering therapists/places by saved GPS city
- Display and matching

So:

- **Landing page:** `CITIES_BY_COUNTRY` (multi-country, display names).
- **Home/dropdown:** `INDONESIAN_CITIES_CATEGORIZED` (Indonesia, with `locationId` for filtering).

Therapist/place **counts per city** are not stored in the app; they come from **Appwrite at runtime**.

---

## 3. How to get therapist and place counts per city

Counts are **live** from your Appwrite database.

### Option A: Script (single city)

```bash
# Count therapists in a city (uses location or city attribute)
APPWRITE_API_KEY=your_key pnpm tsx scripts/query-therapists-appwrite.ts "Yogyakarta" count
# List names
APPWRITE_API_KEY=your_key pnpm tsx scripts/query-therapists-appwrite.ts "Jakarta" list
```

Requires `.env` (or env) with `APPWRITE_API_KEY` and optionally `VITE_APPWRITE_DATABASE_ID`, `VITE_THERAPISTS_COLLECTION_ID`.

### Option B: Location audit (all therapists, correct vs incorrect city)

In the app (or a small Node script that uses the same service):

- Call **`therapistService.getTherapistLocationAudit()`** (from `src/lib/appwrite/services/therapist.service.ts`).

It returns:

- `total`, `liveCount`
- `withCorrectLocation` / `incorrectOrMissingLocation`
- `incorrectDetails`: list of `{ id, name, city }` for therapists whose saved city doesn’t match any app city (locationId/name/alias).

From there you can group by city (e.g. in a script that fetches all therapists and groups by `city` or `locationId`) to get **therapist count per city**.

### Option C: Places

- **Places collection** is currently **disabled** in `appwrite.config.ts` (`collections.places = null`), so the app returns **0 places** and there are no place counts to report until the collection is enabled and used.
